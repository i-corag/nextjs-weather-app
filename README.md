## Next.js and React.js Weather App 🌦️ 🌩️ ☀️

### 🚀 Getting Started:

1. Git clone https://github.com/i-corag/nextjs-weather-app.git
2. install dependencies
   ```sh
   npm i
   ```
3. start the app
   ```sh
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
